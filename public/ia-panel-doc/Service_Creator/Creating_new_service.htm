Creating new service

To create a new service, you can use either Visual Console or Coding Console. 

  1. Click on the "Create New BKC Service" button.
  2. Type a name on the "Name" field.
  3. Choose a category.
  4. Write a description. (It is optional)
  5. Select a logo for you service. (It is optional)
  6. Click on the "Add Service" button. 

 

Note : You can change all of profile information later.
