Device Cards

After installing a new device, BKC Control Panel shows a new device card as follows:

  • Title shows the device name.
  • Icon shows the device type.
  • Type of the Device shows the device type.
  • Battery Indicator shows the battery level of the device.
  • Last Activity Time shows the time of the last activity of the device.
  • Main Functionality activates the main functionality of the device. For example, if you have a IA Light Switch device, you can turn it off/on using this button.
  • Device Menu provides more functionality for the device.
