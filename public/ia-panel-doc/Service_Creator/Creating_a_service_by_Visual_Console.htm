Creating a service by Visual Console

Visual Console is designed to provide an easy toolkit for developing services. On this console a service can be create by interlocking blocks into together.

 

Profile

On the profile, you can see and change the name, category, description and logo of the service.

 

Blocks

On the Blocks tab, there are two sections. 

On the left section, you can browse between blocks and choose everything that you need. Then you can drag and drop the block on the right section.

To interlock the block, bring a block close to another one.

Each block has a special form and color: 

  1. Color indicate the group of block.
  2. Form indicate the type of the block. By the form you can determine which blocks can be interlock together. 

 

Coding

When a block is dragged and dropped on the right section of Blocks tab, a code is written on the Code tab. 

 

 

Payment

There are two field in the Payment tab:

  1. Install Price 
  2. Run Price

 

Publish

‌By clicking on Publish button, you can publish your service on the Service Market.

 

Delete

To delete a service, click on the three-dot button, next to the Publish button. Then click on Delete.
