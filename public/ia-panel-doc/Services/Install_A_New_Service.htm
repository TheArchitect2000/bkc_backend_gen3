Install A New Service

To install a new service follow this steps:

  1. Go to Service Market page from the Left Menu.
  2. Find and select a service and click on it.
  3. In the pop-up card, click on the Install.
  4. Choose a device and click on Activate.

Service Price

A service can install and run both free or by tokens. The price is determined by the provider of the service as follows:

  1. Installing Price is related to the tokens that is needed for installing a service.
  2. Running Price is related to the token that is need to each time service is run.

 

Description Tab

Description is written by the provider to explain service functionality.

 

Specification Tab

Specification is written by provider to list compatibility and accessibility of the service as follows:

  1. Compatibility refers to the devices that is compatible with the service.
  2. Access refers to the different parts of the BKC Control Panel that the service access to them.

 

Comment Tab

On the comment tab, everybody can share ideas about the service.
